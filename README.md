# Screen Recorder Studio Website

Landing page for the Screen Recorder Studio app.

This repository contains a simple static website intended for deployment on Vercel.

## Structure

- `index.html` - page markup
- `styles.css` - visual styling and responsive layout
- `script.js` - lightweight image fallback logic
- `images/` - website image assets

## Local Preview

From the project root, run:

```bash
python -m http.server 4173
```

Then open:

`http://127.0.0.1:4173`
